Udaipur Tourism Website 🗺️
A responsive, visually rich front-end website showcasing travel itineraries, local attractions, and cultural experiences for Udaipur, Rajasthan (The City of Lakes). 
Built entirely with clean, HTML and modern CSS 
Features :
✨Interactive Itineraries: Structured day-by-day travel plans tailored for different trip lengths.
Curated Attractions: Detailed sections for major landmarks like City Palace, Lake Pichola, and Jagmandir.
Responsive Design: Fully optimized for desktops, tablets, and mobile screens.
Modern UI/UX: Features smooth scroll effects, hover animations, and a vibrant color palette inspired by Rajasthani royalty.
Travel Essentials: Integrated quick guides on the best time to visit, local cuisine recommendations, and transport tips.
Prerequisites:
You only need a modern web browser (like Chrome, Firefox, Safari, or Edge) to view this project.
